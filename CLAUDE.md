# Claude Instructions

## Git Workflow

- **Always commit and push every code change**, no exceptions.
- After any new file is created, commit and push it immediately.
- After any edit or addition to existing code, the commit message must be **detailed** and include:
  - What was changed
  - Why it was changed (if known)
  - A summary of every meaningful modification made
- Use `git push` after every commit.
- If the repo has no remote set up, notify the user before pushing.
